# Club MESA PCC Website

> Made with Ruby on Rails

### How do I run the code locally?

#### 1. Clone repo

```
git clone https://github.com/club-mesa-pcc/website.git
cd website
```

#### 2. Install gems

```
bundle
```

#### 3. Run server

```
rails server
```

#### 4. Open browser in browser. Should show on 3000, if not check terminal.

```
http://[::1]:3000
```

> Have fun
